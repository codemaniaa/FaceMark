# FaceMark - AI Powered Attendance Management System

FaceMark is a modern AI-based attendance management system that uses facial recognition technology to automate attendance tracking for educational institutes and organizations. The project provides secure authentication, real-time attendance marking, analytics, reporting, and role-based dashboards using a powerful Django REST API backend and a modern React frontend.

---

# 🚀 Features

## 🔐 Authentication & Authorization
- JWT Authentication
- Secure Login & Registration
- Role-Based Access Control
- Admin, Teacher, and Student Roles
- Password Reset Functionality

## 🤖 AI Face Recognition
- DeepFace Integration
- Real-time Face Detection
- Face Encoding & Matching
- Automatic Attendance Marking
- Camera-Based Verification

## 📊 Attendance Management
- Daily Attendance Tracking
- Present / Absent Status
- Attendance History
- Manual Attendance Override
- Attendance Filtering

## 📈 Reports & Analytics

## 🧑‍💼 Admin Panel

## 🌐 Modern Frontend

- React.js Frontend

## ⚡ Backend Features
